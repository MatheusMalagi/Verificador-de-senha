# 🛡️ Verificador de Intensidade de Senha

Este é um projeto desenvolvido em Python que avalia a segurança de uma senha com base em critérios como comprimento, uso de números, letras maiúsculas/minúsculas e caracteres especiais.

## Demonstração Online
O projeto está hospedado e pode ser testado em tempo real através do link abaixo:
👉 **[Acesse o Verificador de Senha aqui](https://verificador-de-senha-mjm.streamlit.app/)**

## Tecnologias Utilizadas
* **Python**: Lógica central do sistema.
* **Streamlit**: Framework utilizado para criação da interface web.
* **Git/GitHub**: Controle de versão e hospedagem do código.
* **Streamlit Cloud**: Deploy e hospedagem da aplicação.

## Aprendizados
Este projeto foi fundamental para consolidar conhecimentos em:
1. **Desenvolvimento Web com Python**: Transformação de scripts de console em aplicações visuais.
2. **Fluxo de Trabalho Profissional (Git)**: Uso de branches, commits organizados e resolução de conflitos.
3. **Continuous Deployment (CD)**: Configuração de atualização automática entre o repositório e o servidor.

## Como rodar o projeto localmente
1. Clone o repositório: `git clone https://github.com/MatheusMalagi/Verificador-de-senha.git`
2. Crie um ambiente virtual: `python -m venv venv`
3. Instale as dependências: `pip install -r requirements.txt`
4. Execute o app: `streamlit run app.py`
